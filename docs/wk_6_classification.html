6.1.3 Different methods of classification

6.1.3.1 Classification trees

  • classify data into two or more discrete categories When we create a decision tree the final leaves might be a mixture of the categories = impure We quantify this with the Gini Impurity

6.1.3.2 Regression trees

  • predict continuous dependent variable
  • Use the Sum of the Squared Residuals
  • Use the threshold that gives smallest SSR = always the lowest value (to prevent over fitting) = similar to gini impurity *Weakest link pruning

6.1.3.3 Random forest

  • supervised : Pattern recognition or machine learning / Classifier learns patterns in the data / Parametric (normal distribution) or non parametric (not normal)
  • UnSupervised : clustering, k-means, ISODATA, cluster busting
